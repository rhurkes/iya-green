    Iya is an SPA (single page app) for quickly viewing specific weather information and following streams of weather bulletins from official sources. Since it is a web app, it can be used on iOS, Android, OSX, Windows, Linux - as long as you have a relatively recent web browser, Iya will work.

    Supported browsers: Chrome, Firefox, IE10+

    For a quick introduction to Iya and how to use it, please watch the short (2 minute) video below:

    'Iya' is a term from Lakota mythology that represents a storm monster responsible for tornadoes, hurricanes, blizzards, etc.
